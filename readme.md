# is-subtitle

> Check if a filepath is a subtitle file

## Install

```sh
$ npm install --save is-subtitle
```

## Usage

```javascript
var isSubtitle = require('is-subtitle');

isSubtitle('/src/toast.srt');
//=> true

isSubtitle('/src/toast.mp4');
//=> false
```
